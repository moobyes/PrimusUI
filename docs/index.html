简单易用，轻量快捷，为移动端服务的前端框架、组件、工具。

为所有开发者、所有应用场景而设计。

Primus 让前端开发更快速、简单。所有开发者都能快速上手、所有设备都可以适配、所有项目都适用。

通用功能

模块化封装了日常开发中将会用到的通过功能，只需执行简单的require，即可马上使用，感受火箭般的工作效率。

UI按需定制

Primus 的源码是基于最流行的 CSS 预处理脚本 - Sass 开发的。你可以采用预编译的 CSS 文件快速开发，也可以从源码定制自己需要的样式。

全自动构建

采用流行的前端自动化构建工具——Gulp，利用 Node.js 流的威力，你可以快速构建项目并减少频繁的 IO 操作。
