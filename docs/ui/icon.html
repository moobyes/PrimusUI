PrimusUI
  • UI库
  • Gulp

基础样式

  • 文本（typeface）
  • 图标（icon）
  • 网格（grid）
  • 边框（border）
  • 布局（layout）

UI组件

  • 标签（label）
  • 微标（badge）
  • 按钮（button）
  • 列表（list）
  • 表格（table）
  • 表单（form）

UI库中的CSS是通过SASS编译生成。

UI库与flexible.js配合使用。

UI库除了字体、边框宽度等，大部分是使用rem作为单位，可以在gulp中引入插件，自动计算。
