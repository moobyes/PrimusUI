PrimusUI
  • UI库
  • Gulp

基础样式

  • 文本（typeface）
  • 图标（icon）
  • 网格（grid）
  • 边框（border）
  • 布局（layout）

UI组件

  • 标签（label）
  • 微标（badge）
  • 按钮（button）
  • 列表（list）
  • 表格（table）
  • 表单（form）

UI库中的CSS是通过SASS编译生成。

UI库与flexible.js配合使用。

UI库除了字体、边框宽度等，大部分是使用rem作为单位，可以在gulp中引入插件，自动计算。

布局（base/layout.scss）

一、占位（placeholder）

相对居中

默认高宽比为31.25%

重写需覆盖padding-top属性

<div class="ui-placeholder-wrap">
  <div class="ui-placeholder">
    <p>相对居中</p>
    <p>默认高宽比为31.25%</p>
    <p>重写需覆盖padding-top属性</p>
  </div>
</div>

二、留白（whitespace）

留白也就是内边距，分了6档，5px、10px、15px、20px、25px以及30px。

有三种展现方式，全、左右（lr）或上下（tb）内边距。

三、对齐（justify）

1）一行对齐（one）
<ul class="ui-justify">
  <li>
    <div class="demo-img"></div>
  </li>
  <li>
    ......
  </li>
</ul>
2）多行对齐（multi）

多行对齐与上面一样，也是引入“ui-justify”即可。

3）弹性布局对齐（flex）

弹性布局是引入“ui-justify-flex”类。

<ul class="ui-flex ui-justify-flex">
  <li>
    <div class="demo-img"></div>
  </li>
  <li>
    ......
  </li>
</ul>
